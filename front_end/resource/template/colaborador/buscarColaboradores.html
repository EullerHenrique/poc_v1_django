Para usar a api ms-usuario do A.R.MANAGER [Dev], você precisa obter o token.
Devido ao fato do login ser realizado pelo Microsoft Authenticator, não é possível obter o token ao enviar as credencias para o keycloak.
Para obtê-lo, você precisa:
  1. Se conectar a vpn
  2. Logar no sistema
  3. Abrir o dev tools
  4. Ir até a aba application
  5. Ir até a aba session storage
  6. Ir até o campo acess_token
